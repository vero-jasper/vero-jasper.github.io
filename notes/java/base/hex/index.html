进制的转换#

正十进制转换为二进制#

  • 除2取余法，直到商为0时讲余数逆序排序
  • 拆分法，将十进制整数拆分为若干个二进制权重之和，有该权重下面写1，否则写0
$ ... 128 64 32 16 8 4 2 1
$ 0 0 1 0 1 1 0 1
$ 45 => 32 + 8 + 4 + 1 => 101101

正二进制转换为十进制#

  • 加权法，使用二进制中每个数字乘以当前位的权重再累加
$ 0010 1101
$ => 0*2^7 + 0*2^6 + 1*2^5 + 0*2^4 + 1*2^3 + 1*2^2 + 0*2^1 + 1*2^0
$ => 0 + 0 + 32 + 0 + 8 + 4 + 0 + 1
$ => 45

负十进制转换为二进制#

  • 先将十进制的绝对值转换为二进制，然后进行按位取反再加1

负二进制转换为十进制#

  • 先减1再按位取反，合并为十进制整数后添加负号
