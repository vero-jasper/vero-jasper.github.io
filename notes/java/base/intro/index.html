Java简介#

Java最早是由SUN公司（已被Oracle收购）的詹姆斯·高斯林（高司令，人称Java之父）在上个世纪90年代初开发的一种编程语言，最初被命名为Oak，后改名为Java

Java版本#

  • Java SE：Standard Edition
  • Java EE：Enterprise Edition
  • Java ME：Micro Edition
