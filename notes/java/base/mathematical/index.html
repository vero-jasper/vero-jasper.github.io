算术运算

算术运算符#

  • 当两个整数相除时结果只保留整数部分
  • 0不能作为除数
  • + 加法运算符
  • - 减法运算符
  • * 乘法法运算符
  • / 除法法运算符
  • % 取模/取余

关系运算符#

结果一定是boolean类型

  • > 大于            >= 大于等于
  • \< 大于           <= 大于等于
  • == 是否等于  != 是否不等于

自增减运算符#

  • ++表示自增，使当前变量自身数值+1
  • --表示自减，使当前变量自身数值-1

前后++的区别#

++n表示先加1再引用n，n++表示先引用n再+1

public class Plus {
public static void main(String[] args) {
int n = 11;
n++; // 11, 相当于n = n + 1;
n--; // 10, 相当于n = n - 1;
int m = ++n; // 11, 不要这么写
}
}

逻辑运算符#

  • && 逻辑与
  • || 逻辑或

三目运算符#

  • 条件表达式 ? 表达式1 : 表达式2;

赋值运算符#

  • =、+=、-=、*=、/=

移位运算符#

  • << 左移位运算符，用于将数据的二进制向左移动，右边用0补充
  • >> 右移位运算符，用于将数据的二进制向右移动，左边用符号补充
  • >>> 逻辑右移运算符，用于将数据的二进制位向右移动，左边使用0补充

位运算符#

  • & 按位与运算
  • | 按位或运算
  • ~ 按位取反运算
  • ^ 按位异或
