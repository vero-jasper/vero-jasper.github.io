变量的基本概念#

  • 当需要程序记录单个数据内容时，则声明一个变量即可，而声明变量的本质就是在内存中 申请一个存储单元，由于该存储单元中的数据内容可以发生改变，因此得名“变量”。

变量的声明方式#

  • 数据类型 变量名 = 初始值;
$ int age = 18;

标识符的命名法则#

  • 由数字、字母、下划线、$等组成，其中数字不能开头
  • 不能使用java的关键字，如public, static, private, class等
  • 区分大小写，长度没有限制但不宜过长
  • 尽量语义化，支持中文但不推荐使用
