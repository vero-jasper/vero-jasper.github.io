集合类库（一）#

集合的概述#

集合的由来#

  • 当需要java程序记录单个数据内容时，则声明一个变量
  • 当需要java程序记录多个类型相同的数据内容时，声明一个一维数组
  • 当需要java程序记录单个类型不同的数据内容时，创建一个对象
  • 当需要java程序记录多个类型相同的对象数据时，创建一个对象数组
  • 当需要java程序记录多个类型不同的对象数据时，则准备一个集合

集合的框架结构#

  • 顶层框架：java.util.Collection 和 java.util.Map
  • Collection 存取元素的基本单位：单个元素
  • Map 存取元素的基本元素：单对元素

常用方法#

  • boolean add(E e) 向集合中添加对象

  • boolean addAll(Collection<? extends E> c) 将指定元素c添加到集合中

    ...
    Collection c1 = new ArrayList(); // 接口类型的引用指向实现类的对象，形成多态
    c1.add("one");
    c1.add(2);
    c1.add(new Person("vero",18));
    // c1: [one, 2, Person{name='vero', age=18}]
    Collection c2 = new ArrayList();
    c2.add("three");
    c2.add(4);
    c1.addAll(c2);
    // c1: [one, 2, Person{name='vero', age=18}, three, 4]
    ...

  • boolean contains(Object o) 判断是否包含指定对象，工作原理：Object.equals(o, e)

    // equals原理
    public static boolean equals (Object a, Object b) {
    return (a == b) || (a != null && a.equals(b));
    }

  • boolean containsAll(Collection<?> c) 判断是否包含集合c的所有对象

  • boolean retainAll(Collection c) 取集合的交集，并保留到当前集合中（当前集合中的元素没有发生改变返回false，否则为true）

  • boolean remove(Object o) 从集合中删除对象

  • boolean removeAll(Collection c) 删除c集合中的全部元素

  • void clear() 清空集合

  • int size() 返回集合元素个数

  • boolean isEmpty() 是否为空

  • boolean equals(Object o) 是否和o集合相等

  • int hashCode() 获取当前集合哈希码

  • Object[] toArray 将集合转为数组

  • Iterator<E> iterator() 获取当前集合迭代器

Iterator接口（重点）#

基本概念#

  • java.util.Iterator 接口主要用于描述迭代器对象，可以变量Collection集合中的所有元素
  • java.util.Collection 接口继承 Iterator

常用方法#

  • boolean hasNext() 判断集合中收费有有可以迭代的元素
  • next() 查找下一个迭代元素
  • void remove() 删除访问的最后一个元素

for each循环（重点）#

  • 迭代器的简化版
    for(元素类型 变量名: 数组/集合名词) {
    // 循环体
    }

List集合（重中之重）#

基本概念#

  • java.util.List

ArrayList#

  • 访问方便，增删不方便

LinkedList#

  • 访问不方便，增删方便

Stack#

  • 后进先出
  • 采用动态数组进行数据管理

Vector#

常用方法#

  • void add(int index, E element) 向集合中指定位置添加元素
  • boolean addAll(int index, Collection<? extend E> o) 向集合中添加所有元素
  • E get(int index) 获取下标为index的元素
  • int indexOf(Object o) 查找参数指定的对象
  • int lastIndexOf(Object o) 反向查找参数指定的对象
  • E set(int index, Element e) 修改指定位置的元素，返回该位置原有的元素
  • E remove(int index) 删除指定位置的元素，返回该位置原有的元素
  • List<E> sublist(int start, int end) 获取当前集合中下标从start开始到end的子集合[start, end)，子集合和当前集合共用同一块内存空间

Queue集合（重点）#

基本概念#

  • java.util.Queue 与List集合平级
  • 先进先出（队列）
  • 主要实现类是LinkedList，增删房名有优势
