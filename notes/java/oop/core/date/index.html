Date相关#

System类的概念和使用#

基本概念#

  • java.lang.System 类中提供了一些有用的类字段和方法

常用方法#

  • static long currentTimeMills() 返回当前时间与1970年1月1日0时0分0秒之间以ms为单位的时间差

Date类的概述#

基本概念#

  • java.util.Date 类主要用于描述特定的瞬间，可以精确到毫秒

常用的方法#

  • Date() 使用无参构造，返回系统时间
  • Date(long date) 根据参数指定的毫秒数构造对象
  • long getTime() 获取调用对象距离1970.1.1.0:0:0的毫秒数
  • void setTime(long time) 设置调用对象基于基准时间time毫秒的时间点

SimpleDateFormat类的概述#

基本概念#

  • java.text.SimpleDateFormat 主要实现日期和文本之间的转换

常用方法#

  • SimpleDateFormat() 使用无参方式构造
  • SimpleDateFormat(String pattern) 根据参数指定的模式构造对象：y, M, d, H, m, s
  • final String format(Date date) 日期 => 文本
  • Date parse(String source) 文本 => 日期

Calendar#

基本概念#

  • java.util.Calendar 类主要描述特定的瞬间
  • 该类是抽象类，因此不能实例化对象，其具体子类针对不同国家的日历系统，其中最广泛使用的是 GregorlanCalendar， 对应世界上绝大多数国家/地区使用的标准日历系统

常用方法#

  • static Calendar getInstance() 用于获取Calendar类型的引用
  • void set(int year, int month, int date, int hourOfDay, int minute, int second)
  • Date get(Calendar instance)
  • void set(int field, int value) 设置指定字段的数值
  • void add(int field, int amount) 向指定字段增加数值
...
Calendar instance = Calendar.getInstance();
instance.set(2020, 8, 8, 20, 8, 8); // 2020年9月8日20点8分8秒
Date date = instance.getTime();
instance.set(Calendar.YEAR, 2018); // 将instance设置成2018年
Date dt = instance.getTime(); // 转化为Date类型
instance.add(Calendar.MONTH, 2);
Date dt2 = instance.getTime(); // 增加月份后的结果
...

java8日期相关类#

  • Date 类从1900年开始，月份都从0开始
  • 格式化只对 Date 类有用，对Calendar类则不能使用
  • 非线程安全等

概述#

  • java8通过发布Date-Time API来进一步加强对日期和时间的处理
  • java.time 日期/时间的基础API
  • java.time.chrono 提供对不同日历系统的访问
  • java.time.format 格式化和解析日期时间对象
  • java.time.tempora 包含底层框架和扩展特性
  • java.time.zone 支持不同时区以及相关规则的类

LocalDate#

概述#

  • java.time.LocalDate 类主要用于描述年-月-日格式的日期信息，该类不表示时间和时区信息

常用方法#

  • static LocalDate now() 在默认时区中从系统时钟获取当前日期

LocalTime#

概述#

  • java.time.LocalTime 主要用于描述时分秒及纳秒

常用的方法#

  • static LocalTime now() 从默认时区中获取当前时间
  • static LocalTime now(ZoneId zone) 获取指定时区的当前时间

LocalDateTime#

概述#

  • java.time.LocalDateTime 主要用于描述ISO-8601日历系统中没有时区的时间，如2007-12-03T10:15:30

常用的方法#

获取

  • static LocalDateTime now()
  • static LocalDateTime of(int y, int m, int day, int h, int m, int s)
  • int getYear()
  • int getMonthValue()
  • int getDayOfMonth()
  • int getHour()
  • int getMinute()
  • int getSecond()
  • Month getMonth()

设置

  • LocalDateTime withYear(int year)
  • LocalDateTime withMonth(int month)
  • LocalDateTime withDayOfMonth(int day)
  • LocalDateTime withHour(int hour)
  • LocalDateTime withMinute(int minute)
  • LocalDateTime withSecond(int second)

增加

  • LocalDateTime plusYear(long year)
  • LocalDateTime plusMonth(long month)
  • LocalDateTime plusDayOfMonth(long day)
  • LocalDateTime plusHour(long hour)
  • LocalDateTime plusMinute(long minute)
  • LocalDateTime plusSecond(long second)

减去

  • LocalDateTime minusYear(long year)
  • LocalDateTime minusMonth(long month)
  • LocalDateTime minusDayOfMonth(long day)
  • LocalDateTime minusHour(long hour)
  • LocalDateTime minusMinute(long minute)
  • LocalDateTime minusSecond(long second)

Instant#

概念#

  • java.time.Instant 描述瞬间的时间点信息

常用方法#

  • static Instant now() 从系统时钟上获取当前时间 (本初子午线)
  • OffsetDateTime atOffset(ZoneOffset offset) 将此瞬间与offset组合，创建偏移量时间
  • static Instant ofEpochMili(long epochMilli) 根据参数指定的毫秒数构造，参数为距离1970.1.1.0:0:0的毫秒数
  • long toEpochMilli() 获取距1970.1.1.0:0:0的毫秒数

DateTimeFormatter#

概念#

  • java.time.DateTimeFormatter 格式化和解析日期

常用方法#

  • static DateTimeFormatter ofPattern(String pattern) 根据参数指定模式来获取对象
  • String format(TemporalAccessor t) 将参数指定日期时间转换为字符串
  • TemporalAccessor parse(CharSequence text) 将参数指定字符串转换为日期
