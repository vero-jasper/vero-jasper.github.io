String类的概述和使用#

String类的概念#

  • java.lang.String类，用于描述字符串，java程序中所有的字符串面值都可以使用该类的对象加以描述
  • 该类由final关键字修饰，表示该类不能继承
  • 从jdk1.9开始该类底层不再使用char[]存储数据，而是改成byte[]存储，节省了一些空间
  • 该类描述的字符串内容是个常量不可更改，因此可以被共享使用

常量池#

由于String类型描述的字符串内容是常量不可改变，因此java虚拟机将首次出现的字符串放入常量池中， 若后续代码中出现了相同字符串内容则直接使用池中已有的对象而无需申请内存及创建对象，从而提高了性能

...
public static void main(String[] args) {
String str1 = "abc";
String str2 = "abc";
System.out.println(str1 == str2); // 比较地址 true
}
...

常用的构造方法#

  • String() 无参方式构造对象得到空字符序列
  • String(byte[] bytes, int offset, int length) 使用bytes数组下标从offset位置开始的length个字节构造
  • String(byte[] bytes) 使用bytes数组的所有内容构造
  • String(char[] value, int offset, int count) 使用value数组中下标从offset位置开始的count个字符构造
  • String(char[] value) 使用value数组的所有内容构造
  • String(String original) 根据指定的字符串内容来构造副本
...
public static void main() {
String str1 = "hello"; // 1个对象，存放在常量池
String str2 = new String("hello"); // 2个对象，"hello"在常量池，str2指向堆区空间
String str3 = "hello"; // 常量池
String str4 = "hello"; // 常量池
String str5 = new String("hello"); // 堆区
String str6 = new String("hello"); // 堆区
// str3 == str4 // 地址 true
// str3.equals(str4) // 内容 true
// str5 == str6 // 地址 false
// str5.equals(str4) // 内容 true
// str4 == str6 // 地址 false
// str4.equals(str6) // 内容 true
String str7 = "abcd";
String str8 = "ab" + "cd"; // 常量优化机制
String str9 = "ab";
String str10 = str9 + "cd";
// str7 == str8 // 地址 true
// str9 == str10 // false
}
...

常用的成员方法#

  • String toString() 返回字符串本身
  • byte getBytes() 返回byte数组（获取所有字符的ASCII）
  • char[] toCharArray() 返回char数组

String类中字符的获取和使用#

  • char charAt(int index) 获取对应下标的字符
  • int length() 返回字符串指定位置的字符
  • boolean isEmpty() 判断字符串是否为空

String类字符串大小比较#

  • int compareTo(String anotherString) 按字典顺序，比较调用对象和参数对象的大小关系
    • this.charAt(k) - anotherString.charAt(k)
  • int compareToIgnoreCase(String str) 不考虑大小关系

其他方法#

  • String concat(String str) 字符串拼接
  • boolean contains(CharSequence s) 判断当前字符串是否包含参数指定的内容
  • String toLowerCase() 小写
  • String toUpperCase() 大写
  • String trim() 去除前后空格
  • boolean startsWith(String prefix) 是否以参数为开头
  • boolean startsWith(String prefix, int offset) 从指定位置开始是否以参数为开头
  • boolean endsWith(String suffix) 是否以参数为结尾
  • boolean equals(Object anObj) 比较字符串内容是否相等
  • int hashCode() 调用对象的哈希码
  • equalsIgnoreCase(Object anObj) 忽略大小写比较
  • int indexOf(int i)
  • int indexOf(int i, int fromIndex)
  • int indexOf(String str)
  • int indexOf(String str, int fromIndex)
  • int lastIndexOf(int i)
  • int lastIndexOf(int i, int fromIndex)
  • int lastIndexOf(String str)
  • int lastIndexOf(String str, int fromIndex)
  • String substring(int begin, int endIndex) 返回从[begin, endIndex)的字符串
  • String substring(int begin) 返回从begin到末尾的字符串

正则表达式#

本质是一个“规则字符串”，可以用于字符串数据的格式进行验证、匹配、查找、替换等操作。

正则表达式相关方法#

  • boolean matches(String regex) 判断当前正在调用的字符串是否匹配参数指定的正则
  • String[] split(String|regex) 以regex所表示的字符串为分隔符，将字符串拆分为字符串数组
  • String replace(char oldChar, char newChar) 使用参数newChar替换此字符串中出现的所有oldChar
  • String replaceFirst(String regex, String replacement) 替换此字符串匹配给定的正则的第一个字符串
  • String replaceAll(String regex, String replacement) 将字符串中匹配正则的字符串替换成replacement
