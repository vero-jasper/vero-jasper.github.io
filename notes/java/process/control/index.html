if判断#

if (条件) {
// 条件满足执行
}
if (条件) {
...
} else {
// 条件判断为false时，执行else中的语句
}
if (条件1) {
...
} else if (条件2) {
...
} else {
...
}

switch多重选择#

switch (表达式) {
case value1:
...
break;
case value2:
...
break;
default:
...
break;
}
  • 注意：如果遗漏了break将会进入下一个循环，造成严重逻辑错误

while循环#

while (条件) {
... // 循环语句
}

do while循环#

while先判断条件，再执行循环语句 do while则是先执行，再判断，条件不满足时退出

do {
...
} while(条件);

for循环#

int[] arr = new int[10];
for (int i = 0; i < arr.length; i++) {
... // 执行语句
}

break / continue#

while 和 for都可以使用 continue 和 break

break#

循环过程中，使用 break 跳出当前循环

continue#

continue 提前结束本次循环，进入下次循环
